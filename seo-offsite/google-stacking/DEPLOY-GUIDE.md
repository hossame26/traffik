## ÉTAPE 1 — Google Business Profile (15 min)

1. Aller sur google.com/business
2. Créer la fiche "Traffik Web"
3. Catégorie : "Agence web" + "Consultant en marketing internet"
4. Remplir TOUT :
   - Nom : Traffik Web
   - Site : https://traffik-web.fr
   - Téléphone : 06 35 50 53 74
   - Email : laibhossame1@gmail.com
   - Horaires : Lun-Ven 9h-18h
   - Description : (copier la version longue de ninja-linking/TARGETS-CHECKLIST.md)
   - Services : Création Site Shopify, Création Site WordPress, Développement React, SEO, Publicité Digitale
   - Zone : France entière
5. Ajouter 5-10 photos (screenshots du site, portfolio)
6. Publier un premier post Google avec lien vers le blog

**Lien créé :** maps.google.com → traffik-web.fr ✅

---

## ÉTAPE 2 — Google Sites (20 min)

1. Aller sur sites.google.com
2. Créer un nouveau site "Traffik Web — Agence Web France"
3. **Copier-coller le contenu de** `google-site/index.html`
   - Soit en mode visuel (copier le texte + ajouter les liens)
   - Soit en intégrant le HTML via un embed
4. URL : sites.google.com/view/traffik-web
5. Publier le site
6. Soumettre l'URL dans Google Search Console

**Liens créés :** 15+ liens vers traffik-web.fr (services, blog, villes) ✅

---

## ÉTAPE 3 — Google Docs (10 min)

1. Aller sur docs.google.com
2. Créer un nouveau document
3. **Copier-coller le contenu de** `google-docs/guide-creation-site-web-2026.md`
4. Titre : "Guide Complet — Créer Son Site Web en 2026 | Par Traffik Web"
5. Mettre en forme (titres, gras, liens cliquables)
6. Partager → "Accessible à tous avec le lien" → mode "Lecteur"
7. Copier l'URL publique

**Liens créés :** 10+ liens vers traffik-web.fr ✅

---

## ÉTAPE 4 — Google Sheets (10 min)

1. Aller sur sheets.google.com
2. Créer un nouveau classeur
3. **Créer les 5 onglets décrits dans** `google-sheets/prix-sites-web-france-2026.md`
4. Titre : "Baromètre Prix Création Site Web France 2026 — Traffik Web"
5. Mettre en forme (couleurs Traffik, en-têtes, bordures)
6. Chaque onglet a un lien vers traffik-web.fr en footer
7. Partager → "Accessible à tous avec le lien"

**Liens créés :** 5+ liens vers traffik-web.fr ✅

---

## ÉTAPE 5 — Google Slides (15 min)

1. Aller sur slides.google.com
2. Créer une présentation
3. **Créer les 15 slides décrits dans** `google-slides/presentation-traffik-web.md`
4. Template : fond noir, texte blanc, bleu #0066FF
5. Titre : "Traffik Web — Agence Web & Marketing Digital | 2026"
6. Chaque slide avec service a un lien vers la page correspondante
7. Partager → "Accessible à tous avec le lien"

**Liens créés :** 10+ liens vers traffik-web.fr ✅

---

## ÉTAPE 6 — YouTube (20 min)

1. Créer la chaîne youtube.com/@traffikweb
2. **Copier la description de** `youtube/description-chaine.md`
3. Ajouter les 4 liens dans la section "À propos"
4. Upload une vidéo de présentation (même 1 min face cam ou screen recording)
5. Description de la vidéo = bourrée de liens (voir templates)
6. Bannière : image 2560x1440 avec "Traffik Web — Sites Qui Convertissent"

**Liens créés :** 10+ liens vers traffik-web.fr ✅

---

## ÉTAPE 7 — Blogger (15 min)

1. Aller sur blogger.com
2. Créer un blog "Traffik Web Blog"
3. URL : traffikweb.blogspot.com
4. Template : sombre/minimaliste
5. **Publier le premier article depuis** `blogger/article-1-creation-site-web.md`
6. L'article contient 15+ liens vers traffik-web.fr

**Liens créés :** 15+ liens vers traffik-web.fr ✅

---

## ÉTAPE 8 — Inter-linking (10 min)

**CRUCIAL :** Chaque propriété doit linker vers les AUTRES propriétés du stack.

Ajouter dans chaque propriété une section "Retrouvez-nous" :

```
🌐 Site officiel : traffik-web.fr
📍 Google Maps : [lien GBP]
📄 Google Site : sites.google.com/view/traffik-web
📋 Guide création site : [lien Google Docs]
📊 Baromètre prix : [lien Google Sheets]
🎥 YouTube : youtube.com/@traffikweb
📝 Blog : traffikweb.blogspot.com
📽️ Présentation : [lien Google Slides]
```

Ça crée un **réseau de liens internes entre propriétés Google** qui renforce l'ensemble.

---

## ÉTAPE 9 — Indexation (5 min)

1. Aller sur Google Search Console (traffik-web.fr)
2. Soumettre chaque URL publique dans "Inspection d'URL" → "Demander l'indexation"
3. URLs à soumettre :
   - L'URL Google Sites
   - L'URL Google Docs (si indexable)
   - L'URL Blogger
   - L'URL YouTube (chaîne + vidéos)
